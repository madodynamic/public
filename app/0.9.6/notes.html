• Support for multiple homes - select one in the menu.
  • Troubleshooting: if you don't see Apple Home(s) in the menu. Try to uninstall and reinstall the app.
  • Remote control for HomeKit enrolled shades should function as expected.
  • To see our shades UI and remote control please make sure Engineering View in the menu is OFF (unchecked).
  • Commissioning new shade to Apple Home should work as well, but the app may need to reload in order to show newly added shade.
  • As always, if you have any problems - please reach out.
